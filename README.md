# 🍄 GnomeLife Prompt Generator

An interactive, zero-dependency HTML utility built to architect highly detailed prompts for AI image generators (Meta AI, Midjourney, Leonardo AI). Designed with a clean, dark-mode cyberpunk aesthetic, this tool blends classic folklore world-building with modern workflow orchestration.

## ⚡ Features

* **Step-by-Step Construction:** Systematically build character prompts across 12 distinct layers, from core mythology and backstory to exact visual styles and lighting moods.
* **Deterministic Randomization:** Roll random combinations globally or lock specific trait cards to protect your core selections while shifting the surrounding atmosphere.
* **Live Compilation:** Watch your generated prompt update in real-time at the bottom of the screen, formatted cleanly for immediate use.
* **Local-First & Portable:** Entirely self-contained in a single file with zero external tracking, frameworks, or database requirements.

## 🛠️ How to Use

### 1. Run Locally
Because this tool follows a strict local-first philosophy, you don't even need internet access to use it:
1. Download or clone this repository.
2. Double-click `index.html` to launch the generator instantly in your preferred web browser.

### 2. Deploy to Production
To share this interface as a live web tool with your community:
* Connect this repository directly to **Vercel** or **GitHub Pages**.
* The server will automatically detect `index.html` and serve the application instantly with zero configuration required.

## 🎨 Visual Aesthetic & Philosophy

This project is built to support a distinct visual universe—contrasting traditional cozy fantasy folklore with sharp, neon-drenched synth-noir styling (neon pink, purple, and teal palettes mixed with moody, cinematic lighting).
