# 🍄 GnomeLife Prompt Compiler v3.0

An interactive, zero-dependency HTML utility built to architect highly detailed prompts for advanced AI image generators. Designed with a clean, dark-mode cyberpunk aesthetic, this tool blends classic folklore world-building with modern workflow orchestration and deterministic translation pipelines.

---

## ⚡ Features

* **Deterministic Taxonomy Pipeline:** Replaces chaotic keyword stuffing with structured mapping dictionaries to ensure cohesive, reliable prompt composition.
* **Anatomy Boundary Shield (Sanitization Layer):** Automatically translates soft human anatomical traits into precise geometric and textural descriptors (e.g., mapping `High Cheekbones` to `angular skull shape`) to maximize prompt compliance and manage engine bias.
* **Passive Context Conversion:** Dynamically reformats active verbs into environmental and situational context blocks, optimizing how latent diffusion models interpret scene composition.
* **Multi-Architecture Targeting:** Built-in multi-mode optimization weights engineered directly into the output logic:
  * *Flux Mode:* Structured, balanced semantic blocks optimized for raw prompt adherence.
  * *SDXL Mode:* Hierarchical, tiered scalar weights designed for classic emphasis parsing.
* **Step-by-Step Layering:** Systematically build character profiles across 12 distinct layers, from core mythology and backstory to exact visual styles and lighting moods.
* **Deterministic Randomization:** Roll random combinations globally or lock specific trait cards to protect your core selections while shifting the surrounding atmosphere.
* **Local-First & Portable:** Entirely self-contained in a single file with zero external tracking, third-party frameworks, or database requirements.

## 🛠️ How to Use

### 1. Run Locally
Because this tool follows a strict local-first philosophy, you do not need internet access to use it:
1. Download or clone this repository.
2. Open `index.html` directly in your preferred web browser.

### 2. Deploy to Production
To share this interface as a live web tool:
* Connect this repository directly to **GitHub Pages** or **Vercel**.
* The platform will automatically detect `index.html` and serve the application instantly with zero configuration required.
